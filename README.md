# DevRadar
Sistema desenvolvido durante a Décima Semana Omnistack organizada pela Rocketseat.

Verifique os ips e as portas antes de testar. Altere o endereço do mongo para a sua conta.

### Inicialização

backend:
'''
cd backend
yarn dev
'''

web:
'''
cd web
yarn start
'''

mobile:
'''
cd mobile
yarn start
'''
